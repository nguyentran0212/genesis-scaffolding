This directory contains documentations for developers and AI agents working with and using the applications. It has the following main sub-directories:

- `architecture`: describe and explain the architecture of the system to provide context to design and development activities on the codebase.
- `developer_guides`: provide guidelines to do design and development tasks on the codebase. This directory also contains playbook for common development tasks.
- `user_guides`: guidelines for user using the applications

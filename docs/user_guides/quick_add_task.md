# Quick Add Task User Guide

The Quick Add input allows you to create tasks, assign projects, and set dates by typing naturally. The system parses your input in real-time and removes metadata from the final task title.

## Project Tagging
Use the **#** symbol to assign a task to a project.
* Type **#** followed by any part of a project name to see suggestions.
* The system matches the longest project name found in your input.
* Example: `Buy groceries #Personal`

## Dates and Deadlines
The system recognizes natural language dates and times. 

### Planning Date
Type a date without a prefix to set the general assigned date.
* Examples: `today`, `tomorrow`, `Monday`, `Oct 25`

### Hard Deadlines
Use the keywords **due**, **by**, or **deadline** to set a strict deadline.
* Examples: `Submit report due Friday`, `Finish slides by 5pm`

### Scheduled Appointments
Use **at** or **@** to set a specific start time.
* Examples: `Doctor appointment @ 2pm`, `Team sync at 10:30am`

## Keyboard Shortcuts
The input is designed for keyboard-first use.

* **#**: Opens the project suggestion menu.
* **Arrow Up / Down**: Navigates the suggestion list.
* **Enter / Tab**: 
    * If the suggestion menu is open: Selects the highlighted project.
    * If the menu is closed or a full match is typed: Submits the task.
* **Escape**: Closes the suggestion menu.

## Examples
* `Call maintenance tomorrow #Home`
* `Review contract due Wednesday @ 9am #Legal`
* `Pick up dry cleaning today`
* `Update server by midnight #Dev`

## Visual Feedback
Icons and badges appear on the right side of the input bar to confirm which projects and dates have been successfully detected before you submit the task.

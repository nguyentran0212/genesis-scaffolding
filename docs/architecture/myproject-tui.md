# TUI Architecture

## Overview

The TUI (Textual User Interface) is a terminal-based UI built with the Textual framework. It is currently an early-stage stub and not yet operational.

## Current Status

The CLI is the active terminal interface for the application. The TUI is a planned future interface intended to provide a richer interactive experience within the terminal.

## What Exists

A basic Textual application structure has been initialized, but the implementation is not functional.

## What Needs Building

A full TUI implementation is required, including interactive components for agent chat, productivity management, and navigation — similar to the CLI but with Textual's rich UI capabilities.
